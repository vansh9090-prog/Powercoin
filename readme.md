# ⚡ POWER COIN (PWR) - Bitcoin-Style Cryptocurrency

Pure P2P cryptocurrency with Bitcoin-level security and advanced features.

## 🚀 Features

- ✅ **Bitcoin-style Proof of Work** mining (SHA-256 double hash)
- ✅ **21 million total supply** with halving every 210,000 blocks
- ✅ **Pure P2P network** - no central servers
- ✅ **Base58Check addresses** (Bitcoin compatible)
- ✅ **UTXO model** for transactions
- ✅ **Difficulty adjustment** every 2016 blocks
- ✅ **Mempool** for pending transactions
- ✅ **No other tokens** - only PWR
- ✅ **No owner** - truly decentralized
